## 一、协议结构模板（12+1）

```
0. 封面（协议编号、签订日期、签订地点）
1. 协议各方（全称 + 统一社会信用代码 + 品牌名 + 资质）
   鉴于条款（双方资源能力陈述 + 协同逻辑）
   第一条 定义与解释
   第二条 合作内容与模式
   第三条 双方权利与义务
   第四条 费用结算与分润机制
   第五条 知识产权与品牌使用
   第六条 保密条款
   第七条 数据保护与合规
   第八条 违约责任
   第九条 不可抗力
   第十条 协议期限与终止（含试点期考核）
   第十一条 法律适用与争议解决
   第十二条 其他条款
   签署页
```

## 二、5条合规红线（绝对不能碰）

### 红线1：无证经营支付清算（"二清"）
| 风险 | 正确做法 |
|------|----------|
| 平台"统一归集客户资金→再分账" | 改为"通过持牌支付机构/银行合规分账系统完成清分" |
| 法律依据 | 《非银行支付机构监督管理条例》第二条 |

### 红线2：互联网诊疗边界
| 风险 | 正确做法 |
|------|----------|
| "远程预诊"→外籍患者未在国内首诊即获远程诊断 | 改为"远程预检评估与健康咨询（不构成诊疗行为）" |
| 法律依据 | 《互联网诊疗管理办法（试行）》第十六条——不得对首诊患者开展互联网诊疗 |

### 红线3：过境免签适用
| 风险 | 正确做法 |
|------|----------|
| 将"240小时过境免签"写为服务SOP节点 | 改为"来华签证政策咨询与入境便利政策告知"，明确不承诺签证审批结果 |
| 原因 | 过境免签以"过境前往第三国"为要件，专为就医入境的客户不适用 |

### 红线4：海外医疗广告合规
| 风险 | 正确做法 |
|------|----------|
| 仅约束内容真实性，未分配客源国当地合规责任 | 乙方确保推广符合当地法律，违规处罚与索赔由乙方自担 |
| 法律依据 | 《医疗广告管理办法》第三条 + 客源国当地法规 |

### 红线5：博鳌特许药械
| 风险 | 正确做法 |
|------|----------|
| 列入消费医疗L3产品线（特许药械面向疾病治疗，属Phase 2严肃医疗） | L3产品描述改为"须另行签署专项协议后纳入" |

## 三、甲方（平台运营方）利益保护必争条款

### 3.1 反绕开条款（核心资产护城河）
```markdown
合作期内及终止后24个月内：
- 乙方不得引导平台客户绕开甲方平台直接交易
- 乙方不得与甲方已签约医疗机构直接合作同类产品
- 对等设置：甲方不挖乙方海外渠道（提高对方接受度）
- 违约金：绕开交易金额的100%
```
**谈判地位：无让步空间。** 乙方是掌握海外客源渠道的央企旅游集团，一旦跑通"医疗+旅游"模式，甲方的平台价值将被架空。

### 3.2 客户数据归属
- 客户个人信息、交易数据、CRM系统数据 → **甲方单独所有**
- 不得纳入"双方共有知识产权"范围
- 协议终止后30日内乙方返还或删除客户个人信息

### 3.3 保险责任分层（不替乙方"买单"）
| 主体 | 保险责任 |
|------|----------|
| 甲方 | 投保平台责任险，仅覆盖甲方SOP执行偏差 |
| 乙方 | 投保旅行社责任险（法定义务，《旅行社条例》第38条）+ 地接服务责任险，将甲方列为附加被保险人，保单交甲方备案 |
| 医疗机构 | 诊疗损害自行承担（《民法典》第1218条） |

**甲方先行赔付定性**：仅为"客户体验安排"；乙方过失导致的赔付，乙方30日内全额补偿，甲方有权直接从应付佣金中**抵扣**（抵扣权是关键）。

### 3.4 试点期考核（避免3年锁定）
- 生效后前6个月为试点期
- 考核指标：乙方渠道有效订单≥300单 或 医疗体检净销售额≥500万元、渠道转化率达标
- 未达标→任一方可提前30日无责终止
- 试点期内任一方可提前30日无理由终止
- 合作为非排他（保留与其他渠道合作权利）

## 四、商务条款设计规范

### 4.1 机票佣金（注意票款vs服务费）
| 错误 | 正确 |
|------|------|
| 按国际机票净销售额10%计佣 | 按乙方实际收取的机票**服务费**（不含票款）的30%计佣 |
| 原因 | 国际机票票款属代收代付，行业代理佣金率仅1%-3% |

**酒店、用车导游：维持10%**（研究报告支持10%-15%，谈判可争取上调至12%）。

### 4.2 净额结算的发票处理
| 模式 | 描述 |
|------|------|
| 错误 | 净额付款下"收到佣金款项后开票"——增值税链条断裂 |
| 正确 | "全额开票、净额结算"：双方按各自应收佣金全额互开发票，付款按净差额执行 |

### 4.3 渠道归因
- 需在定义条款中明确归因窗口（建议：**90日末次触点**）
- 存量客户识别技术规则需写入附件

| 环节 | 责任方 | 说明 |

## 六、签约主体与资质规范

- **签约名称**：必须使用营业执照全称，品牌名作为补充标注
  - 格式：`【甲方营业执照全称】（品牌名：医路通 CareRoute）`
- **补录字段**：统一社会信用代码、法定代表人
- **资质衔接**：甲方须持有旅行社业务经营许可证（入境旅游业务在旅行社一般许可范围内）  
  如许可证在办→设为协议生效先决条件

## 七、谈判策略

### 必争项（无让步空间）
1. 反绕开条款
2. 保险责任重构中的"乙方过失30日补偿+佣金抵扣权"
3. 资金走持牌机构分账
4. 客户数据归甲方单独所有
5. 主体名称规范+适用法律条款

### 可交换项（用于换取必争项）
1. 非排他→特定客源国/产品线有限排他（需与更高业绩指标对赌）
2. 违约金保底金额（20万→10万对称）
3. 溢价分成比例弹性1-2个百分点
4. 试点期KPI具体数值
5. 酒店/用车导游佣金10%→12%

### 谈判顺序
1. 先抛合规类（资金、远程预诊、过境免签）——"不能签"而非"不想签"
2. 再谈责任重构（保险、数据）
3. 最后谈商务条款（考核、佣金、违约金）

## 九、法律依据速查

| 领域 | 法规 | 要点 |
|------|------|------|
| 旅游 | 《旅游法》第29条 | 入境旅游业务属旅行社一般业务范围 |
| 旅游 | 《旅行社条例》第38条 | 旅行社须投保旅行社责任险（法定义务） |
| 医疗 | 《民法典》第1218条 | 诊疗损害由医疗机构赔偿 |
| 互联网医疗 | 《互联网诊疗管理办法（试行）》第16条 | 不得对首诊患者开展互联网诊疗 |
| 医疗广告 | 《医疗广告管理办法》第3条 | 未取得《医疗广告审查证明》不得发布医疗广告 |
| 数据 | PIPL第38条 | 个人信息跨境提供的法定路径 |
| 支付 | 《非银行支付机构监督管理条例》第2条 | 从事支付业务须取得支付业务许可 |

## 十、公立医院体检中心合作协议设计要点（新增 2026-07-17）

### 10.1 与渠道类协议的本质区别

| 维度 | 国旅（渠道合作） | 体检中心（供应商合作） |
|------|------------------|------------------------|
| 关系性质 | 双向引流，互为渠道 | 平台引客，院方交付 |
| 分润方向 | 双向CPS（各付各的佣金） | 单向：消费者支付价-协议价=平台服务费 |
| 佣金定性 | 渠道推广佣金 | **涉外服务协调费**（非医疗返佣） |
| 合规基础 | 旅游+广告监管 | 体检不在DRG考核范围，返佣合法 |

### 10.2 公立医院体检中心的特殊合规考量

#### 返佣合法性
- 体检中心业务不在DRG/DIP支付改革考核范围内，具备市场化运营空间
- 费用应定性为"涉外服务协调费"或"客源引流服务费"，**绝不出现"医疗返佣"字眼**
- 合同正文明确声明："双方确认本协议为合法的商业合作关系，甲方服务费属于市场化的渠道服务费与客户服务费，不涉及公立医院的医疗服务返佣、药品耗材返利等禁止性安排"

#### 签约主体
- 签约主体可以是"XX医院体检中心"（如医院授权体检中心独立签约）
- 需确认：体检中心是否有独立公章/财务章、独立银行账户
- 如体检中心无独立签约权，需取得医院授权书作为附件

#### 发票处理
- 消费者支付价中的"协议价"部分→院方开医疗发票
- 消费者支付价中的"服务费"部分→平台方开现代服务费发票
- **两票分离**，避免税务风险

### 10.3 服务费比例参考
- 行业惯例：公立医院体检中心渠道服务费通常为**20%-30%**
- L1基础套餐建议20%-25%、L2优享25%-28%、L3尊享28%-30%
- 阶梯越高服务费比例越高（高端客户服务成本更高）
- 可设季度阶梯：月均体检客户≥50人→服务费+2pp

### 10.4 体检中心特有的关键条款

| 条款 | 要点 |
|------|------|
| 专属预约产能 | 承诺每日为外籍客户预留X个体检名额，不接受非客户原因临时取消 |
| 英文陪检 | 体检中心配备英文陪检人员，负责报告解读+注意事项告知 |
| 双语报告 | 中英文PDF+纸质装订版，5个工作日内交付 |
| 反绕开 | 不得引导平台客户绕开平台直接预约（违约金：绕开金额3倍） |
| 未到检费用处理 | 阶梯退款：7日前100%→48h-7日50%→48h内不退款；签证被拒全额退 |

### 10.5 适用模板
- 文件：`医路通×公立医院体检中心_合作协议模板.docx`
- 生成脚本：`generate_hospital_agreement.py`
- 适用对象：华山医院体检中心、协和医院体检中心等同级别机构

## 十一、跨协议付费流程设计（新增 2026-07-17）

### 11.1 核心问题
"医+旅"打包产品的定价天生滞后于交付：体检/齿科服务中可能产生增项，最终账单需在全部服务完成后才能确定。传统"全额定金"模式会导致频繁的跨境补款/退款。

### 11.2 解决方案：预授权 + 分阶段扣款

```
销售时 → 预授权（冻额=公示价×120%，不实际扣款）
  ↓
行前7日 → 首次扣款（机酒+套餐预估，分账到各方）
  → 预授权剩余额度 = 增项buffer
  ↓
服务中 → 增项APP推送+客户确认 → 纳入最终结算
  → 履约前可取消 / 履约后不可退
  → 额度不足90%时自动补授权
  ↓
全部完成后 → 最终扣款（按实际账单）
  → 剩余额度T+1自动释放
  → 仅差额场景产生一次退款
```

### 11.3 预授权 vs 定金的决策依据

| 维度 | 定金（实际扣款） | 预授权（冻结额度） |
|------|-----------------|-------------------|
| 增项处理 | 需客户再次支付→体验差 | 直接纳入最终扣款 |
| 退款 | 跨境退款周期长(3-15工作日) | 额度自动释放(T+1) |
| 外汇波动 | 平台承担 | 客户承担（冻结时锁汇） |
| 心理门槛 | 已付钱→不敢改 | 还没扣→决策自由度高 |
| 适用场景 | 价格确定的标品 | 价格不确定的服务 |

### 11.4 增项退款规则
- **履约前**：客户确认但未执行 → 随时取消，不产生费用
- **履约后**：已实际执行（注射对比剂/进入CT室等） → 不可退款
- **特殊情况**：客户麻醉/不便操作 → 陪同翻译代确认+录音存证，术后2h补签

### 11.5 写入哪些合同

| 合同 | 条款位置 | 内容 |
|------|----------|------|
| 国旅协议 | 2.2.6 | 销售端预授权发起+分阶段扣款+乙方告知义务 |
| 体检中心协议 | 2.2.7 | 增项确认流程+不得直接收费+履约前可取消规则 |
| 院方协议（齿科等） | 待起草 | 同上增项确认流程 |

### 11.6 前置条件
- 银联国际预授权支持"分步扣款+部分释放"——已确认支持
- 持牌支付机构分账系统对接——需落实具体机构
- APP增项确认单推送功能开发——需排期

## 十二、消费医疗机构（齿科/医美/眼科等）合作协议设计要点（新增 2026-07-17）

### 12.1 与体检中心协议的三大核心差异

| 维度 | 体检中心 | 消费医疗机构 |
|------|----------|-------------|
| 治疗周期 | 单次，半天完成 | 分阶段，2周-24个月 |
| 确权节点 | 1次（体检完成后） | **多次（每阶段治疗后）** |
| 付款节奏 | 一次全额预授权 | 分阶段确权+扣款 |
| 增项场景 | 加做检查 | 植骨/上颌窦提升/临时冠等 |
| 质保 | 不适用 | **有（种植体3-5年/正畸效果保证）** |
| 失败处理 | 不适用 | **免费重做条款** |
| 随访周期 | 检后解读即可 | 7天/30天/90天/1年长期 |

### 12.2 API确权链路设计

```
乙方HIS系统 → API → 甲方平台 → 银联系统 → VISA预授权扣款
         ↑                ↑
    治疗阶段完成      客户APP确认
```

**确权触发条件**：
- 标准场景：治疗阶段完成 + 客户APP确认/默示同意
- 默认确权：客户已签署治疗同意书授权分阶段扣款的，后续阶段可默认确权
- 争议保护：客户48小时内可在APP发起确权争议

**时限要求**：
- 阶段完成后24小时内触发确权
- 甲方收到争议后24小时内协调乙方提供记录
- 确权错误需3工作日内退款+更正

### 12.3 分阶段付款比例（种植牙示例）

| 阶段 | 内容 | 建议占比 | 确权触发 |
|------|------|----------|----------|
| 第一阶段 | 初诊+CBCT+方案设计 | 10% | 方案确认后 |
| 第二阶段 | 一期手术（植体植入） | 50% | 手术完成后 |
| 第三阶段 | 二期修复（戴冠） | 40% | 戴冠完成后 |

### 12.4 质保条款设计

| 服务类别 | 建议质保期 | 质保范围 |
|----------|-----------|----------|
| 种植牙（植体） | 3-5年 | 脱落/断裂→免费重做 |
| 种植牙（牙冠） | 2-3年 | 破损/脱落→免费重做 |
| 正畸 | 正畸期+X年 | 复发→免费调整/重新治疗 |
| 近视矫正 | 1-2年 | 视力回退→免费补救手术 |

**关键设计**：
- 质保义务不因协议终止而消除

## 十四、"底价+溢价"定价模式（v4 新增，源自Kimi优化）

### 14.1 核心逻辑
将收费从"固定比例分成"改为"机构固定结算价（底价）+ 平台溢价"：

```
消费者支付价（明码标价一口价）= 机构固定结算价（底价）+ 平台溢价
                                                ↓                    ↓
                                      持牌机构清分→机构账户      清分→平台账户
                                      机构开医疗发票            平台开服务费发票
```

**与资金流的关系**：直清分架构不变，仅清分规则从"按比例"改为"按底价+溢价"。

### 14.2 分类设限（合规核心）

| 机构类型 | 溢价空间 | 限制 |
|----------|----------|------|
| 公立医院体检中心 | ≤底价30% | 涉外特需体检，受特需比例管理+纪检约束 |
| 私立齿科/医美/眼科 | 市场调节价 | 不设硬上限，但明码标价，不得标价外加价 |
| 国旅（机酒文旅） | 不适用 | 维持净销售额分佣 |

### 14.3 三条不可踩的线
1. **不得标价外加价**：《明码标价和禁止价格欺诈规定》第8条——溢价必须含在标价总价里
2. **公立医院溢价不得失控**：溢价过高→医院纪检/价格主管部门介入
3. **不得价格欺诈**：不得虚构原价、虚假折扣、低标高结→外籍客户Chargeback高发点

### 14.4 条款调整要点
- 1.7 服务费定义：从"消费者支付价×X%"改为"消费者支付价 − 固定结算价"
- 第四条：引入"固定结算价（底价）"概念
- 新增溢价管理条款（公立医院设上限，诊所明码标价）
- 附件《服务价格表》锁定各项目固定底价

## 十五、分步签约路线图（源自Kimi）

```
① 国旅长期框架协议（原则+商务留白）
        ↓ ②资金通道必须先定
② 银联框架协议（支付分账通道，分两步签）
        ↓ ③通道+底价是商务落地前提
③ 各机构固定结算价（底价）谈判与签署
        ↓ ④底价定了才能回填
④ 国旅《商务条款补充协议》
```

**关键**：体检、消费医疗两份协议不要等国旅补充协议——底价谈判完成即可签署，避免医疗端被文旅端拖住。

## 十六、资金流与结算标准方案（源自Kimi v1.1-v1.4）

### 16.1 确权两节点
- **服务订立**（到院登记）：核验预授权额度是否足以支付
- **服务完成**：按客户实际接受的服务项目触发确权结算

### 16.2 增项三件套
1. 医疗机构提供《可能增项及价格区间清单》公示
2. 行前双语知情同意签署"增项授权上限"
3. 超限须APP端二次确认；未经确认的增项客户有权拒付

### 16.3 取消与争议
- 已确认增项在实际履约前可随时取消退款
- 已实际实施的项目原则上不退（质量问题除外）
- 完成确权后客户提出质量争议的→暂缓释放对应款项→进入争议流程
